# 💰 经济系统设计文档

> 金币、得分、倍率的完整系统设计

---

## 📊 核心公式

### 最终得分计算
```
最终得分 = 基础分 × 出牌倍率 × 番型倍率 × 成就倍率 × 材质加成
```

### 金币获取计算
```
金币 = 基础金币 + 胡牌金币 + 材质金币 + 剩余行动金币 + 神牌/花牌金币
```

## ✖️ 倍率系统

### 1. 出牌倍率（累积相乘）
| 出牌类型 | 倍率 | 说明 |
|----------|------|------|
| 吃（顺子） | ×1.0 | 不加倍率，但得花牌 |
| 碰（刻子） | ×1.0 | 不加倍率，但得花牌 |
| 杠（四张） | ×3.0 | 加倍率，得花牌+季节牌 |

**示例**：碰→碰→杠→吃 = 1.0 × 1.0 × 3.0 × 1.0 = ×3.0

### 2. 番型倍率
| 番型 | 倍率 | 难度 |
|------|------|------|
| 屁胡 | ×1 | - |
| 平和 | ×1.5 | ⭐ |
| 断幺九 | ×2 | ⭐ |
| 混一色 | ×2.5 | ⭐⭐ |
| 对对和 | ×3 | ⭐⭐ |
| 七对 | ×4 | ⭐⭐⭐ |
| 清一色 | ×5 | ⭐⭐⭐ |
| 连七对 | ×10 | ⭐⭐⭐⭐ |
| 豪华七对 | ×12 | ⭐⭐⭐⭐ |
| 字一色 | ×15 | ⭐⭐⭐⭐⭐ |
| 国士无双 | ×20 | ⭐⭐⭐⭐⭐ |

**番型可叠加**：清一色七对 = ×5 × ×4 = ×20

### 3. 成就倍率
| 成就 | 倍率 | 条件 |
|------|------|------|
| 天胡 | ×10 | 起手直接胡（不出牌不弃牌） |
| 地胡 | ×5 | 只弃牌1次就胡 |
| 人胡 | ×2 | 只弃牌2次就胡 |
| 杠上开花 | ×3 | 杠后摸的牌完成胡牌 |
| 海底捞月 | ×2 | 用最后1次出牌机会胡牌 |
| 速战速决 | ×2 | 2次出牌内胡牌 |
| 完美主义 | ×3 | 不使用任何弃牌机会 |

### 4. 材质倍率加成
| 材质 | 倍率加成 |
|------|----------|
| 铜牌参与的面子 | +0.5 |
| 玻璃牌参与的面子 | ×2（一次性） |
| 幸运牌 | 30%概率 ×2 |

## 💡 爽感设计原则

1. **数字要大**：后期倍率 ×100+ 很正常，看着爽
2. **反馈明确**：每次加成都有动画和数字跳动
3. **连击感**：多个倍率依次叠加，一个个弹出
4. **音效配合**：大倍率要有震撼音效
5. **意外惊喜**：幸运牌的30%暴击、随机金神等
6. **成长感**：从 ×3 到 ×100 的build成长曲线
