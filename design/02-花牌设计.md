# 🌸 花牌设计文档

> 花牌：出牌时获得，主动使用，一次性消耗

## 获取规则

| 出牌类型 | 花牌选择 |
|----------|----------|
| 吃（顺子） | 2选1 |
| 碰（刻子） | 3选1 |
| 杠（四张） | 4选1 |

## 花牌分类

| 花色 | 主题 | 图标 |
|------|------|------|
| 🌸 梅 | 弃牌相关 | 梅花 |
| 🎋 竹 | 摸牌/牌库 | 竹子 |
| 🌺 兰 | 倍率/得分 | 兰花 |
| 🏵️ 菊 | 牌面转化 | 菊花 |

## 稀有度

| 稀有度 | 出现权重 | 效果强度 |
|--------|----------|----------|
| 普通 | 50% | 小增益 |
| 稀有 | 35% | 中等增益 |
| 史诗 | 15% | 强力/有条件 |

---

## 🌸 梅牌（弃牌系）

### 普通
| 名称 | 效果 |
|------|------|
| 梅开一度 | +1 弃牌次数 |
| 梅开二度 | +2 弃牌次数 |
| 疏影横斜 | 本次弃牌上限 +2 |
| 暗香浮动 | 本次弃牌，弃掉的牌会告诉你它们在牌库的位置 |

### 稀有
| 名称 | 效果 |
|------|------|
| 傲雪凌霜 | 下次弃牌不消耗弃牌次数 |
| 梅花三弄 | 接下来3次弃牌，每次弃完后额外摸1张（立刻弃掉保持14张） |
| 踏雪寻梅 | 弃牌时可以指定想要摸到的花色 |
| 一剪梅 | 弃掉1张牌，从牌库中精确找出1张指定的牌 |

### 史诗
| 名称 | 效果 |
|------|------|
| 梅花落 | 弃掉所有手牌，重新摸14张（不消耗弃牌次数） |
| 寒梅傲骨 | 本回合弃牌次数用完后，仍可继续弃牌（每次 -20 基础分） |

---

## 🎋 竹牌（摸牌/牌库系）

### 普通
| 名称 | 效果 |
|------|------|
| 翠竹引路 | 看牌库顶3张，选1张加入手牌（弃1张保持14张） |
| 竹报平安 | 看牌库顶5张牌 |
| 节节高升 | 把牌库中所有该花色的5变成6（或6变7...） |
| 竹林听风 | 听到下3张将要摸到的牌的花色 |

### 稀有
| 名称 | 效果 |
|------|------|
| 青竹直上 | 选中1张牌，从牌库找出与它相邻的牌（如3万→找2万或4万） |
| 移花接木 | 选中1张牌，与牌库顶的牌交换 |
| 胸有成竹 | 下次弃牌，100%摸到你需要的牌（选1种） |
| 势如破竹 | 连续摸3张牌，然后弃3张 |

### 史诗
| 名称 | 效果 |
|------|------|
| 竹林七贤 | 从牌库中找出7张任意牌，与手牌交换7张 |
| 雨后春笋 | 复制选中的1张牌，永久加入牌库（牌库 +4 张该牌） |

---

## 🌺 兰牌（倍率/得分系）

### 普通
| 名称 | 效果 |
|------|------|
| 幽兰吐芳 | 倍率 +0.3 |
| 兰心蕙质 | 基础分 +30 |
| 空谷幽兰 | 下次出牌倍率 ×1.5 |
| 蕙质兰心 | 本回合花牌效果 +50% |

### 稀有
| 名称 | 效果 |
|------|------|
| 金兰之交 | 倍率 +1.0 |
| 芝兰玉树 | 基础分 +100 |
| 兰桂齐芳 | 倍率 ×1.5（本回合永久） |
| 玉兰花开 | 如果下次胡牌是清一色，倍率 ×3 |

### 史诗
| 名称 | 效果 |
|------|------|
| 兰亭集序 | 本回合所有倍率翻倍 |
| 采兰赠芍 | 牺牲所有倍率加成，换取等量金币 |

## 🏵️ 菊牌（转化系）

### 普通
| 名称 | 效果 |
|------|------|
| 菊花一朵 | 选中1张数字牌，+1或-1 |
| 秋菊傲霜 | 选中1张牌，变成同花色的1或9 |
| 黄花晚节 | 选中1张牌，与另一张选中的牌交换数字 |
| 采菊东篱 | 随机将1张手牌变成字牌 |

### 稀有
| 名称 | 效果 |
|------|------|
| 菊花台 | 选中最多3张牌，全部变成同一个数字 |
| 持菊问道 | 选中1张牌，变成任意指定的牌 |
| 金菊绽放 | 选中1张牌，变成金牌材质 |
| 秋色满园 | 所有1和9变成5 |

### 史诗
| 名称 | 效果 |
|------|------|
| 满城尽带黄金甲 | 所有手牌变成金牌材质 |
| 菊残犹有傲霜枝 | 选中3张牌销毁，从牌库任选3张替换（牌库永久 -3） |

## 特殊组合（隐藏效果）

同时持有特定花牌会触发组合效果：

| 组合 | 条件 | 效果 |
|------|------|------|
| 梅兰竹菊 | 4种花色各1张 | 全部效果 ×2 |
| 双梅 | 2张梅牌 | 弃牌次数 +1 |
| 双竹 | 2张竹牌 | 可以多看2张牌库 |
| 双兰 | 2张兰牌 | 倍率额外 +0.5 |
| 双菊 | 2张菊牌 | 转化不消耗花牌 |

---

## 设计原则

1. **保持14张**：所有效果都不会让手牌+已出牌超过或少于14张
2. **有选择**：同类型花牌有多种选择，不同局有不同最优解
3. **可组合**：花牌之间可以配合，形成连招
4. **不必须**：不用花牌也能胡牌，花牌是锦上添花
