# 胡！🀄

一款 Roguelike 麻将卡组构建游戏，灵感来自 Balatro。组牌、算番、收集神牌，向着最终的「胡」进发！

## 概念

将 Balatro 令人上瘾的 Roguelike 卡组构建玩法，用麻将机制重新演绎。不再是扑克牌型，而是麻将番型；不再是小丑牌，而是神牌——以各种疯狂的方式改变你的得分。

## 玩法

### 核心循环
1. **出牌**：你有 14 张麻将牌，组成有效的面子+雀头来得分
2. **计分**：根据你组成的番型计算分数——越难的番型分数越高
3. **过关**：达到目标分数进入下一轮
4. **商店**：回合间购买神牌、花牌和升级
5. **循环**：难度逐渐增加，你能走多远？

### 核心系统

- **番型系统** — 麻将牌型作为得分阶梯（类似 Balatro 中的扑克牌型）
  - 鸡胡 → 平胡 → 对对胡 → 混一色 → 清一色 → 七对 → 国士无双...
  - 每种番型有基础 chips × 倍率

- **神牌** — 被动效果卡（类似小丑牌）
  - 🀀 东风大师：手牌含风牌时倍率 ×2
  - 💎 清一色专家：同花色牌 chips +30
  - 🌙 月下美人：七对子倍率 ×3
  - 🔥 杠上开花：有杠子时触发额外得分

- **花牌** — 消耗品（类似塔罗牌/星球牌）
  - 转化手牌中的牌
  - 升级特定番型的基础分
  - 重新摸牌

- **Boss 关卡** — 特殊限制
  - "不能使用万子"
  - "所有牌面朝下"
  - "只能用对子胡牌"
